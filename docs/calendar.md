## Live Family Calendar

!!! success "Interactive Calendar - Add & Edit Events"

    **How to Add Events:**

    1. Click on any date in the calendar below
    2. Sign in with your Google account (if prompted)
    3. Add event details (title, time, location, description)
    4. Save the event - it will appear for all family members!

    **How to Edit Events:**

    - Click on any existing event to view details
    - If you have permission, you can edit or delete events
    - Contact Yuanxi or Jingya to request calendar edit access
